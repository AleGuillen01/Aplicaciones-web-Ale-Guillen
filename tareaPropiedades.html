Tabla de Propiedades CSS Utilizadas

Nombre de propiedad Función Posibles valores Otros datos relevantes
color Define el color del texto Colores (red, blue), HEX (#ff0000), RGB (rgb(255, 0, 0)) Hereda valores del elemento padre
background-color Establece el color de fondo del elemento Colores, HEX, RGB Si no se define, el fondo es transparente
font-size Define el tamaño del texto px, em, rem, %, small, medium, large Valores relativos dependen del elemento padre
margin Espaciado exterior de un elemento px, %, auto Auto se usa para centrar elementos
padding Espaciado interno de un elemento px, %, em Define espacios alrededor del contenido
border Define el borde del elemento px (ancho), tipo (solid, dashed, dotted), color Puede combinarse con border-radius
display Determina cómo se muestra un elemento block, inline, flex, grid, none None oculta el elemento
